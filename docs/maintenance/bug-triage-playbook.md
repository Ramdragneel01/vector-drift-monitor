# Bug Triage Playbook

Repository: vector-drift-monitor
Last updated: 2026-05-05

## Intake
- Capture reproduction steps, expected behavior, and actual behavior.
- Record environment details and severity.

## Prioritization
- Critical: production outage, data risk, or severe regression.
- High: major user impact with workaround.
- Medium/Low: limited scope or minor friction.

## Resolution
- Assign owner, estimate fix scope, and add test coverage notes.
- Close only after verification in target environment.
