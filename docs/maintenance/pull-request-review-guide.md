# Pull Request Review Guide

Repository: vector-drift-monitor
Last updated: 2026-05-05

## Reviewer Checklist
- Validate functional correctness and edge-case handling.
- Confirm tests and lint checks are sufficient.
- Check for security, performance, and maintainability risks.

## Author Checklist
- Keep PR focused and include rationale in description.
- Add screenshots or logs when behavior changes.
- Include rollback or mitigation notes for risky changes.
