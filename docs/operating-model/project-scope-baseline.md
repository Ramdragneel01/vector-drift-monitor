# Project Scope Baseline

Repository: vector-drift-monitor
Default branch: main
Last reviewed: 2026-05-05

## Scope Statement
- Maintain a reliable and testable codebase for the repository purpose.
- Keep documentation, quality checks, and release hygiene consistent.
- Improve contributor onboarding and change-review clarity.

## In Scope
- Product and platform features maintained in this repository.
- Build, test, lint, and release automation tied to this repository.
- Documentation updates that reduce operational ambiguity.

## Out Of Scope
- Cross-repository process changes owned by other projects.
- Unplanned breaking changes without explicit release notes.
